# Java Mini Projects 

A curated collection of Core Java projects showcasing object-oriented programming, file handling, arrays, loops, methods, exception handling, and interactive console applications.

---------------------------------------------------------------------------------------

Projects Included:-

> Quiz Game

An interactive quiz application featuring:

* Player name input
* 10 multiple-choice questions
* Score calculation
* Percentage calculation
* Pass/Fail status
* Performance evaluation

---------------------------------------------------------------------------------------

> Note Taking Application

A file-based note manager with:

* Add notes
* View notes
* Delete all notes
* Date and time stamp
* Persistent storage using File Handling

---------------------------------------------------------------------------------------

> Tic Tac Toe Game

An advanced multiplayer Tic Tac Toe game featuring:

* Dynamic board sizes (3×3, 4×4, 5×5)
* Player names
* Winner detection
* Draw detection
* Replay option
* Scoreboard
* Input validation
* Multiple rounds

---------------------------------------------------------------------------------------
 

> Currency Converter

A command-line currency converter application that allows users to convert currencies between different countries.

Features:
* Convert between USD, INR, EUR, GBP, and JPY
* View supported currencies
* Menu-driven interface
* Object-Oriented Design
