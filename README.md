# Java Mini Projects 

A curated collection of Core Java projects showcasing object-oriented programming, file handling, arrays, loops, methods, exception handling, and interactive console applications.

---------------------------------------------------------------------------------------

Projects Included:-

> Weather Forecast Application

A real-time weather application using APIs and JSON parsing.

Features:
* Real-time weather data using API
* City-based search
* JSON parsing
* Clean CLI interface
* Wind speed and temperature display
* Condition analysis (Cold / Pleasant / Hot)

Features:
